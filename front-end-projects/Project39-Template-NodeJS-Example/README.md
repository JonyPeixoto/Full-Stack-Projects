  ## :information_source: How to Use

## Here, you will find the step by step to run this project:

OBSERVATION: If you have to change the directory of your SSD/HD, use the cd /d command (DESIRABLE DIRECTORY FOLLOWED BY ":") - without quotation marks. For example:

I switched disk C to disk D:

cd /d D:

1) Download, install and start [xampp](https://www.apachefriends.org/pt_br/index.html) and press START on servers [Apache](https://www.apache.org/) and [ MySQL](https://www.mysql.com/)

2) Download and install [NodeJS](https://nodejs.org/en/docs)

If you have difficulties with [nodemon](https://www.npmjs.com/package/nodemon) install it using the -g flag as follows:

npm install -g nodemon

3) To start your project, go to your project folder. To do so, launch CMD or command prompt as an administrator and follow the sequence below:

cd (PROJECT FOLDER PATH, CREATE A FOLDER CALLED "NODE_PROJECT" - WITHOUT THE QUOTES)

npm init

Populate the entry point: app.js

And all other fields, leave blank.

<br/>

Apply the command below to install all dependencies:

npm install express

I created the “index.js” and all the variables

Finally, go to your command prompt and type:

node index.js

4) Install MySQL

npm install MySQL

5) Install the browser that renders the pages and the ejs dependency:

npm install body parser

npm install ejs

Finally, go to your command prompt and type:

nodemon index.js or launch LIVE in VS Code

Done, the browser will open and you will see the print screen that I shared here in the README!
